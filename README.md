# OHS Player

## Project Description

OHS Player is a media player application built with Java/Kotlin. This project aims to provide a clean and efficient media playback experience.

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or higher
- Gradle (for building the project)

### Installation

```bash
# Clone the repository
git clone https://github.com/ohs-foundation/ohs-player.git

# Navigate to the project directory
cd ohs-player

# Build the project
gradle build
```
